你为什么要学 Git

你应该会编程吧。你有没有在写大项目的时候，反复修改某段代码？在你修改之前，可能会备份当前的版本，以免之后再想改回去。但每次修改之前都备份未免太过繁琐，所以你需要一个版本控制系统来管理代码的每个版本。Git 就是最流行的版本控制系统。

Git 的历史

Git 是为管理 Linux 项目开发的。要管理一个有几千万行代码的项目，Git 必须非常强大。它既要在备份上万个文件时足够快，又要满足开发者的不同需求。

另外，由于 Linux 有很多人参与，所以 Git 必须支持大量开发者同时工作。这也催生了 GitHub 这种 Git 平台。全世界约 80% 的代码托管在 GitHub 上，所以如果你喜欢写代码就很有必要了解 Git 和 GitHub。

你应该怎么学 Git

你可以通过很多方式学习 Git。最常见的方法是读 Pro Git。但是如果你不想看书，你可以接着看这个网站。
